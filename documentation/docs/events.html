رویداد ها

لورم ایپسوم متن ساختگی با تولید سادگی نامفهوم از صنعت چاپ، و با استفاده از طراحان گرافیک است، چاپگرها و متون بلکه روزنامه و مجله در ستون و سطرآنچنان که لازم است، و برای شرایط فعلی تکنولوژی مورد اصلی، و جوابگوی سوالات پیوسته اهل دنیای موجود طراحی اساسا مورد استفاده قرار گیرد.

DayaChat_sendMessage

با استفاده از این رویداد می توانید وقتی کاربر پیامی ارسال میکند، به محتوای پیام دسترسی پیدا کنید.

 document.addEventListener("DayaChat_sendMessage", (e) => {
    let message = e.detail.message;
    ... 
    //do somthing with message
 })
توضیح parameter type
متن پیام ارسالی detail.message string

DayaChat_getMessage

با استفاده از این رویداد می توانید وقتی کاربر پیامی دریافت میکند، به محتوای پیام دسترسی پیدا کنید.

 document.addEventListener("DayaChat_getMessage", (e) => {
    let message = e.detail.message;
    let sender = e.detail.sender;

    ... 
 })
توضیح parameter type
متن پیام دریافتی detail.message string
فرستنده پیام detail.sender string
